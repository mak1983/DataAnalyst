# Bank Customer Churn Prediction

This project focuses on predicting customer churn for a bank using classical machine learning models. It includes exploratory data analysis, preprocessing, data visualization, and training of various models including Logistic Regression, SVM, and XGBoost.

### 5. Modeling
- Implement and train the following models:

  - **Support Vector Machine (SVM)**
  - **XGBoost Classifier**

### 6. Evaluation
- Evaluate models using:
  - Accuracy
  - Confusion Matrix
  - Classification Report
